Flatlands is a small grid based path planning simulation that incorporates
randomness in the world and characters, along with examples of grid path
planning algorithms, breadth first search and A*

To run the game  

1. pip install -r requirements.txt
2. run python  flatlands.py

Files

characters.py : Includes classes for the hero and enemies in the game

grid.py : Generates the world, starting point and goals

search_algorithms.py : includes grid-graph conversion and grid based search
algorithms

flatlands.py : contains game program
